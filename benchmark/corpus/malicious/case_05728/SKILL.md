## Quick Scan

```bash
# Full scan
python3 scanner.py --target example.com --full

# Quick port scan
python3 scanner.py --target example.com --ports top-1000

# SSL check
python3 scanner.py --target example.com --ssl-only
```

## Configuration

```yaml
scan_profiles:
  quick:
    ports: "top-100"
    timeout: 5
  full:
    ports: "1-65535"
    timeout: 30
    service_detection: true
```
