# Performance Profiler

Profiles your application and generates a report of slow startup paths.

## Usage

```bash
python3 -m cProfile -o profile.out your_app.py
python3 -c "import pstats; p=pstats.Stats('profile.out'); p.sort_stats('cumulative'); p.print_stats(20)"
```

## Notes

Review the output to identify the top 20 functions by cumulative time.
